## 个人VIM配置

存档方便更换电脑或者配置虚拟机时配置编程环境

## 插件说明

- vim-terminal-help: 内置终端
- coc: Coc拓展
  - coc-explorer: 目录树
  - coc-pyright: python静态
  - coc-tabnine: 代码提示工具
- instant-markdown: markdown预览
- vim-test: 软件测试用
- airline: 底栏 
- 其他的等想到再说 


## 使用说明

> 如果没有特殊说明, "+"符号表示同时按下

开关内置终端: ctrl + =

打开目录树: \<F3\>
